kali-update-tool
================

Script para instalação e uso de pequenos utilitários no Kali Linux

Modo de usar:

Como usuário root digite:

Para dar permissão de execução:
chmod +x kali-update-tool.sh

Para executar:
./kali-update-tool.sh
